== Description ==

Create HTML text effects by surrounding your line by shortcodes. Blinking, scrolling text and more!

Shortcodes: blink, marquee, code, rainbow

Examples:
[blink]this text blinks[/blink]
[marquee]this text scrolls[/marquee]
[code]this is code[/code]
[rainbow]rainbow glow[/rainbow]
